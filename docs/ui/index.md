# User Interface Subsystem

The **User Interface Subsystem** manages screen layouts, player selection menus, passcode entry forms, in-game HUD displays, pause menus, and end-of-game victory screens for *Lan Master*.

---

## Screen Modules
* [Main Menu](mainmenu.md) - Title screen navigation, option toggles, music selection, and 4-digit passcode entry interface.
* [In-Game Pause Menu](gamemenu.md) - In-game contextual overlay menu allowing players to resume gameplay, restart the current level, or return to the main menu.
* [Victory Screen](welldone.md) - Game completion screen featuring animated text routines, sound effect triggers, and background graphics.
* [Menu Subsystem Overview](menus.md) - Unified architectural analysis of screen state transitions and UI rendering loops.

## Formatting Utilities
* [UI Formatting Utilities](ui_formatting.md) - Assembly helpers for formatting timer numbers, terminal counts, passcode strings, and ASCII characters into PPU tile attributes.
