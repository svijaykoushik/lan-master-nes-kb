# Lan Master NES – Instruction Manual (2011)

## Overview
- **Purpose:** Run the game on Nintendo Entertainment System (NES) / Famicom hardware or an emulator.
- **Hardware required for console play:** A flash cartridge (e.g., PowerPak).
- **License:** Public domain, free of charge, provided *as is* with no warranties.

## Gameplay
1. **Goal:** Connect all computers on each level using wire fragments.
2. **Controls:** Rotate wires/computers with **A**, **B**, or **Select** buttons.
3. **Time limit:** Each level is timed; running out ends the game.
4. **Pass codes:** After completing a level a pass code is shown in the pause menu. It can be entered later via the **Code** item to resume from that level, and it is automatically applied when a new level starts.
5. **Levels:** 50 levels in total. Later levels introduce more complex networks, additional wire types, unused segments, and gradual network degradation.

## Russian (Русский) – краткое содержание
- Игра предназначена для NES/Famicom или их эмуляторов. Требуется флеш‑картридж (PowerPak) для реальной консоли.
- Задача: соединить все компьютеры, используя отрезки провода и кнопки **A**, **B**, **Select**.
- Ограничение по времени; при истечении времени игра заканчивается.
- После уровня выдаётся код, который можно ввести в главном меню через пункт **Code**.
- Всего 50 уровней, сложность растёт: новые типы проводов, неиспользуемые фрагменты, деградация сети.
