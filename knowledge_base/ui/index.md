# Assembly Modules
* [Gamemenu](gamemenu.md) - A contextual menu allowing players to resume the game, restart the level, or exit.
* [Mainmenu](mainmenu.md) - The primary user interface for game start, passcode entry, and audio options.
* [Menus](menus.md) - Analysis of the main menu, in-game pause menu, and win screen.
* [UI Formatting](ui_formatting.md) - Utility routines for rendering numbers, tables, and passcodes to the PPU.
* [Welldone](welldone.md) - Logic for the final win screen, including animated text and background effects.
